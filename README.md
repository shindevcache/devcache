# Shin DevCache

Shin DevCache lets user store and read code snippets

## Installation

Use the [npm](https://www.npmjs.com/) to install packages.

```bash
npm install
```

## Usage

```bash
npm run dev
```
## Troubleshooting

You need **dotenv** file to establish connection to the database.
